# Support

## Getting Help

- Usage questions: open a GitHub issue with context and reproducible steps.
- Bugs: use the Bug Report issue template.
- Feature requests: use the Feature Request issue template.
- Security concerns: follow [SECURITY.md](SECURITY.md).

## What to Include

- Operating system and Python version
- Exact command/config used
- Relevant logs and stack traces
- Minimal reproduction steps

## SLA Notes

PQTS is community-maintained. Response times vary by complexity and maintainer
availability.
